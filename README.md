# Ping Pong Pro

Un juego de Ping Pong con control por gestos y físicas avanzadas.

## Características

*   Modo de un jugador vs IA.
*   Control por gestos usando la cámara.
*   Simulación de físicas avanzadas.
*   Sistema de puntuación.
*   Menú interactivo.

## Instalación

1.  Clona este repositorio.
2.  Instala las dependencias desde el archivo `requiriments.txt`:
    ```
    pip install -r requiriments.txt
    ```
3.  Ejecuta el juego:
    ```
    python run.py
    ```

## Controles

*   **Espacio:** Pausar/Sacar
*   **R:** Reiniciar la pelota
*   **ESC:** Volver al menú
*   **G:** Activar/Desactivar el control por gestos
*   **F:** Activar/Desactivar la visualización de físicas
*   **Teclas de flecha:** Mover la paleta en 2D (cuando el control por gestos está desactivado)

## Tecnologías Utilizadas

*   Python
*   Pygame
*   OpenCV
*   MediaPipe
